# The Rice Field Application | Na Kaow

## Project Overview

This project was conducted as part of the **Artificial Intelligence Fundamentals** course, a core subject in the second year of study. The course required students to design an application that integrates artificial intelligence to support real-world problem solving.

My group was assigned the topic of **Agriculture**, and through group brainstorming and discussion, we decided to develop a **rice disease detection application** for Thai farmers. The project involved collecting datasets from multiple sources, including online image repositories, publicly available datasets on Kaggle, previously taken photographs, and field data collection.

These datasets were used to train and develop a machine learning model for rice disease detection, which was later integrated into the designed application called **“Na Kaow”**.

## My Role

- Served as the **team coordinator**, responsible for task distribution and internal communication within the group  
- Participated in **data collection and data preparation**  
- Worked on **model development and parameter tuning**, focusing on the *False Smut rice disease* dataset  
- Prepared and refined the dataset before passing it to the main dataset developer for further integration and processing  
- Responsible for **writing, organizing, and formatting the final project report** for submission to the course instructor
